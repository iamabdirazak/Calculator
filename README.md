# Calculator

A simple calculator web application built with React, TypeScript, and Vite.
It supports basic arithmetic operations, including addition, subtraction, multiplication, and division.
The interface displays the current input, the previously entered value, and the selected operator.
All calculator logic is handled through a dedicated state hook for clarity and separation of concerns.
